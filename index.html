Animation Framework

Smooth Anims

Declarative scroll animations for any website. Zero dependencies, ~3 KB gzipped.

Animation Presets

Add data-animate to any element. It reveals on scroll.

Fade

Simple opacity transition.

data-animate="fade"

Fade Up

Rises into view from below.

data-animate="fade-up"

Fade Down

Drops in from above.

data-animate="fade-down"

Fade Left

Slides in from the right.

data-animate="fade-left"

Scale

Grows from 92% to full size.

data-animate="scale"

Blur Up

De-blurs while rising.

data-animate="blur-up"

Rotate

Subtle rotation + scale.

data-animate="rotate"

Hover Lift

Micro-interaction on hover.

data-hover="lift"

Staggered Lists

Animate children sequentially with data-animate-stagger.

  • First item — 0 ms delay
  • Second item — 100 ms delay
  • Third item — 200 ms delay
  • Fourth item — 300 ms delay
  • Fifth item — 400 ms delay

Timing & Easing

Control duration, delay, and easing per element.

Snappy · 300 ms

Quick and responsive.

Smooth · 800 ms

Default feel — gentle deceleration.

Spring · 600 ms

Slight overshoot for playfulness.

Quick Start

Drop two files into your project and add data attributes. 

<link rel="stylesheet" href="css/smooth-anims.css">
<script src="js/smooth-anims.js"></script>

<div data-animate="fade-up">Hello world</div>

<ul data-animate-stagger="fade-up"
    data-animate-stagger-delay="80">
  <li>One</li>
  <li>Two</li>
</ul>

<script>
  SmoothAnims.init(); // call once on page load
</script>
